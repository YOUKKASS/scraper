Why This Is Hard To Build

Each platform required a completely different technical approach.

🔄

Remote Selector Updates

When sites change their DOM, update one GitHub JSON file — all users fixed instantly. No republishing needed.

🔒

MV3 Compliant

Built natively for Manifest V3. MV2 extensions are being deprecated. Most competitors haven't migrated yet.

🌐

Shadow DOM Traversal

AliExpress uses Web Components. This extension traverses Shadow DOM trees — most tools fail silently here.

⚛️

React Store Bypass

Temu blocks inline scripts via CSP. Uses chrome.scripting in MAIN world — the only reliable MV3 solution.

🪟

Dual Layout Support

AliExpress has two layouts: full page and mini popup. Detects which is active and handles each correctly.

🧩

Modular Architecture

One file per platform. Clean separation. Easy to extend or sell individual modules separately.
